## Equipamento Principal — PISOM BB Laser

### O que é

O **PISOM BB Laser** da Medical San é um equipamento de alta performance que combina duas tecnologias laser complementares em um único aparelho, com uma única ponteira:

| Tecnologia | Comprimento de onda | Principal benefício |
|---|---|---|
| Thulium | 1927nm | Renovação celular, efeito glow, uniformização |
| Erbium | 1550nm | Rejuvenescimento, manchas, firmeza, cicatrizes |

### Diferenciais Clínicos

- Software inteligente com protocolos modernos
- Alta precisão e operação intuitiva
- Uma única ponteira para múltiplos protocolos
- Resultados premium com alta performance

### Especificações Técnicas

| Especificação | Detalhe |
|---|---|
| Comprimento de onda | 1550nm · 1927nm |
| Largura de pulso | 0,06 ~ 8ms/dot |
| Densidade | 25 ~ 3025 PPA/cm² (12 níveis) |
| Área de aplicação | 20 × 20mm |
| Laser de mira | < 2mW / 650nm |
| Registro ANVISA | 82338020002 |
| Alimentação | Bivolt automático 100–240V |
